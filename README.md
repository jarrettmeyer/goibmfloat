# goibmfloat

[Golang](https://golang.org/) project to convert IBM floats to IEEE floats.

## Usage

Get the project.

```
$ go get github.com/jarrettmeyer/goibmfloat
```

Use the project in your project

```
package myproject

import (
    "github.com/jarrettmeyer/goibmfloat"
)
```

## Testing

```
$ go test
```
